сверстайте адаптивный лендинг по макету в Figma https://disk.yandex.ru/d/xYAclHtuJVlChg", 
используя стек html + css + чистый js (без библиотек и фреймворков).

   * Придерживайтесь принципа Pixel Perfect.
   * Страница должна быть адаптивной, 
хорошо выглядеть при любом размере экрана 
(
нет горизонтального скролла, 
нет уезжающих за экран текстовых элементов, 
текст не наезжает на другой текст или изображение
)
   * Избегайте дублирования текстового контента в мобильной и десктопной версиях.

   * Добавьте корректно работающую бегущую строку.

   * Кнопки на стартовом экране являются якорями и 
ведут к соответствующим блокам.

   * Карусель с карточками участников должна быть зацикленной 
и меняться автоматически через каждые 4 секунды.

   * Карусель с этапами не должна быть зацикленной 
и не должна автоматически менять слайды.

   * Добавьте анимацию по своему усмотрению.
