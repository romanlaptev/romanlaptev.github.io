install

https://nodejs.org/en/download/releases/
https://unix.stackexchange.com/questions/207591/how-to-install-latest-nodejs-on-debian-jessie

- скачать бинарный архив tar.gz
https://nodejs.org/dist/latest-v8.x/node-v8.17.0-linux-x86.tar.gz

- распаковать в /home/bin
- переименовать в node

- создать символьную ссылку
ln -s /home/nodejs/bin/node node

- проверить версию
node -v
v8.17.0
